# 15 Puzzle Game

A classic sliding puzzle game built with TypeScript, React, and Next.js.

## How to Play

Arrange the numbered tiles in order (1, 2, 3...) by clicking on tiles adjacent to the empty space. Only tiles next to the empty square can be moved.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Features

- Responsive grid layout
- Click-to-move tile mechanics
- Randomize button to shuffle the puzzle
- Win detection with congratulations message
- Update the config file to decide if you want more rows or columns. 

Enjoy solving the puzzle!
